## 📝 使用说明

### 开发环境
```bash
npm run dev          # 启动开发服务器
npm test             # 运行测试
npm run test:watch   # 监听模式测试
```

### 测试流程
1. 访问 http://localhost:3000
2. 注册新用户（输入昵称）
3. 创建房间（设置房间名和密码）
4. 其他用户加入房间（输入密码）
5. 用户互相给分
6. 查看实时分数更新
